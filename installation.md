# Installation Guide

## Step 1
Head to the releases page of Elden Ring Nightfarer's Revenge here

Download the .zip file anywhere on your desktop.

## Step 2 
Open the file directory of your Elden Ring Nightreign Game:
(:X/folder_name/steam/steamapps/common/Elden Ring Nightreign/game)
Alternatively you can just browse local files of the game through Steam.

Open the .zip file you just downloaded and drag the ELDEN VIS NIGHTREIGN folder into the game directory (same directory with nightreign.exe)
